About LettrIt
LettrIt is a simple application, that let's the user input usual information, needed to writa a formal letter. It then supplies the user with a Print button, that will return a beautiful layouted letter document ready for printing.
This app is a community project, manly developed by @frosch03 and @jobacca. You can find the source of LettrIt on github.
